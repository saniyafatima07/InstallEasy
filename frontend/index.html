InstallEasy

Windows Installation Guide

1. Open PowerShell as Administrator

Right-click on the Windows Start button and select "Windows PowerShell (Admin)" or "Command Prompt (Admin)"

2. Install the Tool

Run the following command:

npm install -g cpp-env-setup

3. Run Installation Script

install-cpp-windows

4. Wait for Installation

The script will automatically install:

  • Chocolatey package manager
  • MinGW C++ compiler
  • Visual Studio Code
  • Required VS Code extensions

5. Verify Installation

g++ --version

Linux Installation Guide

1. Open Terminal

Open your system terminal

2. Verify Sudo Access

sudo -v

3. Install the Tool

sudo npm install -g cpp-env-setup

4. Run Installation Script

install-cpp-unix

5. Verify Installation

g++ --version

macOS Installation Guide

1. Open Terminal

Open Terminal from Applications > Utilities

2. Install the Tool

npm install -g cpp-env-setup

3. Run Installation Script

install-cpp-unix

4. Verify Installation

clang++ --version

Important Note

After installation, restart your terminal to ensure all PATH changes are applied. If you encounter any issues, please check the troubleshooting section or report the issue on our GitHub repository.
